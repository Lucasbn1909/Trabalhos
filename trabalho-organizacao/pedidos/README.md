# Pedidos

Fluxo recomendado: receber, validar pagamento/status, separar, conferir, embalar,
emitir documentos na ferramenta autorizada, despachar, acompanhar entrega e tratar
exceções.

Nunca salve no Git detalhes de compradores, etiquetas, notas fiscais ou comprovantes.
Use `pedidos/dados/` apenas como área local ignorada quando uma ferramenta operacional
mais adequada ainda não estiver disponível.
