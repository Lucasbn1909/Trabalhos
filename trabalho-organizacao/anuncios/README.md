# Anúncios

Organize aqui padrões reutilizáveis de títulos, descrições, atributos, categorias e
checagem de qualidade. Dados exportados das plataformas e cópias de anúncios reais
devem permanecer em `anuncios/exportacoes/`, que é ignorada.

Antes de publicar, confira: SKU, categoria, atributos obrigatórios, preço, estoque,
dimensões, peso, garantia, variações, imagens, prazo e regras vigentes da plataforma.
