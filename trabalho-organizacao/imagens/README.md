# Imagens

As imagens operacionais são separadas localmente em `originais/`, `editadas/` e
`publicadas/`; essas subpastas estão ignoradas para evitar arquivos grandes e material
com metadados indevidos no Git.

Antes de usar uma imagem, confira direitos de uso, enquadramento, fundo, legibilidade,
fidelidade do produto, metadados, nomes de arquivo e requisitos atuais da plataforma.
Somente documentação ou ativos leves, deliberadamente aprovados, devem ser versionados.
