# Automações

Área reservada a scripts e integrações futuras. Cada automação deverá documentar:
objetivo, entrada, saída, permissões, agendamento, logs, retentativas, limites das APIs,
modo de teste, reversão e responsável.

Não há integração implementada. Não preencha credenciais neste repositório; use
variáveis de ambiente e mantenha apenas os nomes em `.env.example`.
