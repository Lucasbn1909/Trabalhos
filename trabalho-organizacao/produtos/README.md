# Produtos

Documente aqui o padrão de SKU, atributos de catálogo, regras de custo, preço e
estoque. Cadastros e exportações reais devem ficar em `produtos/dados/`, ignorada pelo
Git.

Campos mínimos sugeridos: SKU, EAN/GTIN quando aplicável, nome, marca, categoria,
variação, custo, preço, estoque, peso, dimensões, fornecedor e status. Defina a fonte
oficial de cada campo antes de automatizar sincronizações.
